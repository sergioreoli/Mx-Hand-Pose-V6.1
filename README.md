For Armature generate site Mixamo.com or Armature generation in addon "Mixamo Control Rig"

> **Professional hand pose control for Mixamo armatures in Blender**

## 🎯 Quick Start

1. [Download](https://github.com/sergioreoli/mx-hand-pose/releases) the add-on
2. Install in Blender (`Edit > Preferences > Add-ons > Install`)
3. Select your armature
4. Open sidebar (`N`) and go to **"Mx Hand Pose"** tab
5. Start posing!

## ✨ Features

| Feature | Description |
|---------|-------------|
| 🖐️ **Full Hand Control** | Control all 5 fingers with simple buttons |
| 👍 **Individual Thumb Control** | Manipulate each of the 3 thumb bones separately |
| 🎯 **20+ Pre-made Poses** | Ready-to-use gestures (Peace, OK, Point, etc.) |
| 🔄 **Pose Mirroring** | Instant mirror from left to right hand |
| 📹 **Keyframe Support** | Insert keyframes directly from the panel |
| 🤲 **Wrist Rotation** | Full XYZ wrist control with rotation buttons |
| 🎚️ **Closure Slider** | Smooth transition from open to closed fist |
| 🔧 **Mixamo Compatible** | Works with Mixamo armatures and Control Rig |

## 📦 Installation

### Automatic (Recommended)
1. Download Unzip file  `unzip_addon_mx_hand_v6_1.zip` or`addon_mx_hand_v6_1.py`
2. Blender → Edit → Preferences → Add-ons
3. Click **Install** → Select file → Enable add-on

### Manual
Copy to your Blender add-ons folder:
- **Windows**: `%APPDATA%\Blender Foundation\Blender\3.x\scripts\addons\`
- **Linux**: `~/.config/blender/3.x/scripts/addons/`
- **macOS**: `~/Library/Application Support/Blender/3.x/scripts/addons/`
